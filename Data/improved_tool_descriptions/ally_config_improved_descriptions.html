New Tool Description
Retrieve the full conversation history of a specific session for a given Copilot by providing its session ID and endpoint identifier. Use this tool when you need to access detailed session data, including all messages exchanged during that session. It fits into workflows involving session analysis, review, or troubleshooting by allowing you to fetch complete session records for inspection or processing.
